A modern single-page application built with Vite and React that provides an intuitive interface for interacting with Spotify data through a custom backend API.

## Overview

This frontend application serves as the user interface for the Spotify integration platform, offering a fast and responsive experience powered by Vite's lightning-fast build tooling and React's component-based architecture.

## Features

- **Single Page Application**: Seamless navigation without page reloads
- **Modern UI**: Clean, intuitive interface for Spotify interactions
- **Responsive Design**: Works across desktop, tablet, and mobile devices
- **API Integration**: Connects to custom Express.js backend

## Tech Stack

- **Build Tool**: Vite
- **Framework**: React
- **Language**: JavaScript/JSX
- **Styling**: CSS
