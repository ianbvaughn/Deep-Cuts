# Deep Cuts
This is the documentation for all data pertaining to [Deep Cuts](https://github.com/ianbvaughn/Deep-Cuts), a full-stack web application built on Express.js and Supabase.

This project uses a 'medallion' schema architecture, with the following schemas:

- `bronze`: Raw data
- `silver`: Cleaned data
- `gold`: Cleaned and transformed data, ready for analysis

Project created and owned by Ian Vaughn (ianbvaughn@gmail.com)
